先ほどの作品の釈文です。<br/>
<br/>
<br/>
<br/>
紫陌紅塵拂面來<br/>
<br/>
無人不道看花回<br/>
<br/>
玄都観裏桃千樹<br/>
<br/>
盡是劉郎去後栽<br/>
<br/>
・・・・・・・・・・・<br/>
<br/>
紫陌の紅塵面を払って来る<br/>
<br/>
人の花を看て回ると道わざるなし<br/>
<br/>
玄都観裏桃千樹<br/>
<br/>
尽く是れ劉郎去って後に栽える<br/>
<br/>
・・・・・・・・・・・<br/>
<br/>
都大路に吹きつける塵に顔を掩う<br/>
<br/>
塵を立てる人混み、花見の帰りだ<br/>
<br/>
玄都観に咲く一千本もの桃の木は<br/>
<br/>
劉君が立ち去った後に植えたのだ<br/>
<br/>
<br/>
雰囲気で。<br/>
だいたいで。<br/>
こんなかんじで。<br/>
